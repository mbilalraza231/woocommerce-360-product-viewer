# WP 360 Viewer

A simple WordPress plugin that adds a 360-degree product rotation viewer using a shortcode.

## Usage

Add the shortcode to a post or page:

```php
[wp360 images="image1.jpg,image2.jpg,image3.jpg"]
```

## Features

- Enqueues plugin JavaScript and CSS
- Uses a shortcode for easy insertion
- Loads a 360 viewer from a list of images
